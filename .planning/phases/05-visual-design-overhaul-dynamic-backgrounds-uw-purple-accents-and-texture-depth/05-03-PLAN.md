**ProjectCard.tsx** -- Add CardSpotlight as inner visual wrapper:

1. Import CardSpotlight from '@/components/effects/CardSpotlight'
2. The CardSpotlight goes INSIDE the outer motion.div (which has `layout` prop), NOT outside it
3. This is critical: the spotlight effect only manipulates a CSS mask on an overlay div, it does NOT affect the card's box model or position, so it won't interfere with Motion's layout calculations
4. Wrap the inner content (the motion.div with layout="position") with CardSpotlight:
```tsx
<motion.div
  layout
  className={`rounded-xl bg-white shadow-lg overflow-hidden ${
    isExpanded ? 'col-span-1 md:col-span-3' : ''
  }`}
  whileHover={{
    y: -2,
    boxShadow: '0 8px 30px rgba(0, 0, 0, 0.12)',
  }}
  transition={{
    layout: layoutTransition,
    ...{ duration: 0.3, ease: easing.out },
  }}
>
  <CardSpotlight
    radius={300}
    color="rgba(75, 46, 131, 0.12)"
    className="h-full"
  >
    {/* Existing motion.div layout="position" with all card content -- unchanged */}
    <motion.div layout="position" className="cursor-pointer" onClick={onToggle}>
      {/* ... collapsed/expanded content ... */}
    </motion.div>
  </CardSpotlight>
</motion.div>
```
5. Do NOT modify the expanded/collapsed content structure, the onClick handlers, or the layout animation props
6. The spotlight color "rgba(75, 46, 131, 0.12)" is UW purple at 12% opacity for a subtle glow

**Anti-patterns to avoid:**
- Do NOT put CardSpotlight OUTSIDE the motion.div with layout prop (breaks layout animation)
- Do NOT add position: relative/absolute to CardSpotlight if it would change the card's box model
- Do NOT modify the LayoutGroup in ProjectsSection.tsx

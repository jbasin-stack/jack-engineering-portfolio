**Fix 2: Lenis scroll jitter (GAP-05)**

The problem: `data-lenis-prevent` on the Embla viewport disables ALL Lenis processing inside the element, including vertical wheel events. When the cursor is over the carousel and the user scrolls vertically, native browser scroll and Lenis fight (one handles it, the other does not), causing jitter.

The fix: Remove `data-lenis-prevent` entirely. Embla uses pointer events (drag/touch) for carousel navigation, NOT wheel events. Lenis processes wheel events for vertical page scroll. They operate on different event types and do not conflict. The `touchAction: 'pan-y pinch-zoom'` on the flex container is sufficient for mobile -- it tells the browser to treat vertical gestures as page scroll and only pass horizontal gestures to Embla.

Replace the viewport div:
```tsx
{/* OLD: */}
<div ref={emblaRef} className="overflow-hidden" data-lenis-prevent>

{/* NEW: */}
<div ref={emblaRef} className="overflow-hidden" style={{ overscrollBehaviorX: 'contain' }}>
```

Keep `touchAction: 'pan-y pinch-zoom'` on the flex container unchanged.

**Fix 3: Update PROJ-05 test**

Since `data-lenis-prevent` is removed, update the test in `src/components/projects/__tests__/carousel.test.tsx`:

Find and replace the PROJ-05 test:
```tsx
// OLD:

});

import { motion } from 'motion/react';
import type { Project } from '../../types/data';
import { easing } from '../../styles/motion';

interface CarouselCardProps {
  project: Project;
  onClick: () => void;
}

export function CarouselCard({ project, onClick }: CarouselCardProps) {
  return (
    <motion.div
      className="group cursor-pointer rounded-xl bg-card shadow-md overflow-hidden h-full border border-transparent transition-colors duration-300 hover:border-accent/30"
      whileHover={{
        scale: 1.02,
        boxShadow: '0 8px 32px rgba(0,0,0,0.12), 0 0 0 1px oklch(0.55 0.15 250 / 0.15)',
      }}
      transition={{ duration: 0.3, ease: easing.out }}
      onClick={onClick}
    >
      {/* Featured accent bar */}
      {project.featured && (
        <div className="h-1 w-full bg-gradient-to-r from-accent/80 via-accent to-accent/80" />
      )}

      {/* Thumbnail with gradient overlay and hover zoom */}
      <div className="relative overflow-hidden">
        <img
          src={project.thumbnail}
          alt={project.title}
          className="w-full aspect-video object-cover transition-transform duration-500 group-hover:scale-105"
          loading="lazy"
          decoding="async"
        />
        <div className="absolute inset-0 bg-gradient-to-t from-black/20 via-transparent to-transparent" />
      </div>

      {/* Text content */}
      <div className="p-5">
        <h3 className="text-lg font-bold text-ink leading-tight">{project.title}</h3>
        <p className="text-sm text-silicon-600 mt-2 line-clamp-2 leading-relaxed">{project.brief}</p>

          {project.domain}
        </span>
      </div>
    </motion.div>
  );
}
```

**Sort projects:** Featured first, then non-featured. Use `[...projects].sort((a, b) => Number(b.featured) - Number(a.featured))` to create `sortedProjects`.

**Embla setup:**
```typescript
const [emblaRef, emblaApi] = useEmblaCarousel({
  align: 'start',
  containScroll: 'trimSnaps',
  dragFree: false,
  loop: false,
  slidesToScroll: 1,
  duration: prefersReducedMotion ? 0 : 25,
});
```

**Arrow button state:**
- `canScrollPrev` and `canScrollNext` state (both `useState<boolean>`)
- `useCallback` handlers: `scrollPrev = () => emblaApi?.scrollPrev()`, `scrollNext = () => emblaApi?.scrollNext()`
- `useEffect` that subscribes to `emblaApi.on('select', updateButtonState)` and `emblaApi.on('init', updateButtonState)` where `updateButtonState` reads `emblaApi.canScrollPrev()` and `emblaApi.canScrollNext()`
- Clean up with `emblaApi.on('destroy', ...)` or return cleanup in useEffect

**Dot indicators (mobile only):**
- `selectedIndex` state tracking current slide: `emblaApi.on('select', () => setSelectedIndex(emblaApi.selectedScrollSnap()))`
- `scrollSnaps` state: `emblaApi.scrollSnapList()` called on init
- Render dots as small circles below the carousel, `className="flex md:hidden justify-center gap-2 mt-4"`
- Each dot: `button` with `className="h-2 w-2 rounded-full transition-colors"`, active: `bg-accent`, inactive: `bg-silicon-200`
- `onClick={() => emblaApi?.scrollTo(index)}`

**JSX structure:**

Create `src/components/ui/AnimatedTabs.tsx`:
- Props: `tabs: { id: string; label: string }[]`, `activeTab: string`, `onChange: (id: string) => void`
- Renders a `div` with `role="tablist"` and rounded-xl container with subtle background (`bg-silicon-50/50 dark:bg-silicon-200/10 p-1 backdrop-blur-sm`)
- Each tab is a `button` with `role="tab"`, `aria-selected`, `aria-controls={panel-${tab.id}}`
- Active tab renders a `motion.div` with `layoutId="active-tab-indicator"` positioned absolutely behind the button text (z-index -1), styled as `bg-white/80 dark:bg-white/10 shadow-sm rounded-lg`
- Spring transition: `type: 'spring', stiffness: 400, damping: 30`
- Tab buttons are `flex-1` for equal width, text-sm font-medium
- Active tab text: `text-ink`, inactive: `text-silicon-600 hover:text-ink` with `transition-colors`

Create `src/components/sections/Expertise.tsx`:
- Static domain merge mapping object (per locked decisions):
  - `fabrication`: skillDomain 'Fabrication', toolCategory 'Fabrication Processes'
  - `rf-test`: skillDomain 'RF', toolCategory 'Lab Equipment'
  - `analog`: skillDomain 'Analog', toolCategory 'EDA Tools' (excluding Xilinx Vivado)
  - `digital`: skillDomain 'Digital', tools: ['Xilinx Vivado'] (extracted from EDA Tools)
- Build tabs array from the mapping (labels: Fabrication, RF & Test, Analog, Digital)
- Default active tab = first domain in the derived tabs array (data-driven per locked decision)
- Section wrapper: `section` with `id="expertise"`, `role="region"`, `aria-label="Expertise"`, standard padding `px-6 py-24`
- Use `sectionVariants` and `fadeUpVariant` from `styles/motion.ts` for section entry animation
- Section title: `h2` with "Expertise" text, `text-2xl font-bold text-ink`
- Render `<AnimatedTabs>` below the heading
- Tab content wrapped in `AnimatePresence mode="wait"`:
  - `motion.div` with `key={activeTab}`, `role="tabpanel"`, `id={panel-${activeTab}}`
  - Variants: initial `{ opacity: 0, scale: 0.96, filter: 'blur(8px)' }`, animate `{ opacity: 1, scale: 1, filter: 'blur(0px)' }`, exit same as initial
  - Transition: `duration: 0.25, ease: easing.out` (from styles/motion.ts)
  - Glassmorphic panel classes: `rounded-xl backdrop-blur-md bg-white/10 dark:bg-white/5 border border-white/20 dark:border-white/10 p-6 mt-6`
  - Two-column layout inside: `grid grid-cols-1 md:grid-cols-2 gap-8`
  - Left column: bold sub-heading "Skills" (`text-sm font-bold text-ink uppercase tracking-wide mb-3`), then `ul` of skill items styled as `text-silicon-600`
  - Right column: bold sub-heading "Tools & Equipment" (same style), then `ul` of tool items
  - Add a `min-h-[200px]` to the panel container to prevent layout jump between tabs (Pitfall 4 from research)

**SVG Path (TIME-01):**
- Replace the `motion.div` scaleY line with an SVG element positioned absolutely on the left side of the timeline
- SVG contains two `<path>` elements sharing the same `d` attribute (vertical line):
  1. **Undrawn track:** `stroke="var(--color-silicon-200)"`, `strokeWidth={1}`, `strokeDasharray="4 4"`, `fill="none"` -- a faint dashed line showing the full path
  2. **Drawn progress path:** `stroke="url(#timeline-gradient)"`, `strokeWidth={2}`, `fill="none"`, `style={{ pathLength: scrollYProgress }}` via `motion.path`
- SVG `<defs>` block with `<linearGradient id="timeline-gradient">` -- accent-600 at top (`stopOpacity="1"`) to accent-400 at bottom (`stopOpacity="0.6"`). Use CSS custom properties: `var(--color-accent)` for stroke colors
- SVG: `className="absolute left-0 top-0 h-full w-6"`, `viewBox="0 0 24 {totalHeight}"` where the path is `M 12 0 V {totalHeight}`. Use `preserveAspectRatio="none"` so the SVG stretches to fill the container height
- `useScroll` with `target: containerRef, offset: ['start 0.8', 'end 0.6']` (same as current)

**TimelineNode component (TIME-02, TIME-03, TIME-04):**
- Props: `milestone: TimelineMilestone`, `threshold: number` (index / (total - 1)), `scrollYProgress: MotionValue<number>`
- **One-shot activation (replaces per-frame setState):**
  - `const [hasActivated, setHasActivated] = useState(false)`
  - `useMotionValueEvent(scrollYProgress, 'change', (latest) => { if (!hasActivated && latest >= threshold) setHasActivated(true); })` -- fires setState exactly once per node, never resets
- **Continuous content animation (no setState):**
  - `const contentOpacity = useTransform(scrollYProgress, [threshold - 0.05, threshold + 0.02], [0, 1])`
  - `const contentY = useTransform(scrollYProgress, [threshold - 0.05, threshold + 0.02], [8, 0])`
  - Wrap milestone content in `motion.div` with `style={{ opacity: contentOpacity, y: contentY }}`
- **Node circle styling:**
  - Inactive: `border-2 border-silicon-200 bg-cleanroom` (hollow circle)
  - Active: `border-accent bg-accent shadow-[0_0_12px_oklch(0.55_0.15_250/0.4)]` (filled with ambient glow)
  - Size: `h-3 w-3 rounded-full`, positioned `absolute -left-[21px] top-1` (centered on the SVG path at x=12)
  - `transition-all duration-300` for smooth state change
- **One-shot pulse ring (TIME-03):**
  - When `hasActivated` becomes true, render a `<span>` inside the node div with class `pulse-ring`
  - Add CSS `@keyframes` for `pulse-ring` in a `<style>` tag or Tailwind arbitrary animation:
    - `0%`: `transform: scale(1); opacity: 0.3`
    - `100%`: `transform: scale(2.5); opacity: 0`
  - Animation: `animation: pulse-ring 1.5s ease-out forwards` (one-shot via `forwards` + `animation-iteration-count: 1`)
  - The span is `absolute inset-0 rounded-full border-2 border-accent/30`
  - Use Tailwind `@keyframes` via `animate-[pulse-ring_1.5s_ease-out_forwards]` if possible, or define the keyframes in the component file using a top-level style injection or inline `<style>` tag scoped by a unique class name

**Container layout:**
- Same outer structure: `section#timeline > div.mx-auto.max-w-5xl > h2 + div.relative.mt-12.pl-8`
- Container `ref={containerRef}` on the inner div with `relative` positioning
- Adjust `pl-8` if needed so content aligns to the right of the SVG path (pl-10 or pl-12 may be needed with the wider SVG)

**Performance note:** The key optimization is replacing the old pattern (8 `setIsActive(latest >= threshold)` calls per scroll frame) with `useTransform` for continuous values and one-shot `useState` for discrete activation. This reduces per-frame re-renders from 8 to 0 during normal scrolling.

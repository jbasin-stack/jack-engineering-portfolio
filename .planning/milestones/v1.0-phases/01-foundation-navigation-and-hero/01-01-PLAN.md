   - Use `useRef<LenisRef>` to reference the Lenis instance

   - Return `<ReactLenis root options={{ autoRaf: false, lerp: 0.1, duration: 1.2, smoothWheel: true, ...(!reducedMotion ? {} : { enabled: false }) }} ref={lenisRef}>{children}</ReactLenis>`

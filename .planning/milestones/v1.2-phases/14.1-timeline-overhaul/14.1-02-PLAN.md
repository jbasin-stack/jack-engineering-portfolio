    Complete replacement of `src/components/sections/Timeline.tsx`. Remove ALL existing code (SVG path, useScroll, useTransform, useMotionValueEvent, TimelineNode subcomponent). Write from scratch.

    **Component structure:**

    1. **Layout variant system:**
       ```typescript
       type LayoutVariant = 'large' | 'half' | 'overlay';
       const LAYOUT_CYCLE: LayoutVariant[] = ['large', 'half', 'overlay'];
       ```
       Each entry gets variant via `LAYOUT_CYCLE[index % LAYOUT_CYCLE.length]`.

    2. **Year extraction:**
       ```typescript
       function extractYear(date: string): string {
         const match = date.match(/\d{4}/);
         return match ? match[0] : date;
       }
       ```

    3. **Animation variants (inline, not imported from motion.ts — these are timeline-specific):**
       - `entryVariants`: parent container, staggers children with `staggerChildren: 0.12`
       - `fadeUp`: child elements, `hidden: { opacity: 0, y: 24 }` to `visible: { opacity: 1, y: 0, duration: 0.5, ease: easing.out }`
       - `dotVariant`: dot markers, `hidden: { opacity: 0, scale: 0 }` to `visible: { opacity: 1, scale: 1, duration: 0.4, ease: easing.out }`
       - Import `easing` from `../../styles/motion` for consistent curves.

    4. **TimelineEntry subcomponent:**
       ```typescript
       function TimelineEntry({ milestone, index }: { milestone: TimelineMilestone; index: number })
       ```
       - Receives milestone data and index for variant calculation.
       - Wraps in `<motion.article>` with `variants={entryVariants}`, `initial="hidden"`, `whileInView="visible"`, `viewport={{ once: true, amount: 0.2 }}`.
       - Generous padding: `className="relative py-16 md:py-20"` (per user decision: py-16 to py-20).
       - Left padding `pl-10 md:pl-14` for connector line space.

       **Entry content (all wrapped in motion.div with fadeUp variant):**
       - **Dot marker:** Absolutely positioned on the left edge of the connector line. `absolute left-0 top-20 md:top-24 h-[6px] w-[6px] -translate-x-1/2 rounded-full bg-accent`. Uses `dotVariant`. Add `data-testid="timeline-dot"`.
       - **Year:** `<motion.span>` with `text-5xl md:text-7xl font-bold text-accent/80` — large, accent-colored, slightly transparent for elegance.
       - **Title:** `<motion.h3>` with `mt-3 text-xl md:text-2xl font-semibold text-ink`.
       - **Description:** `<motion.p>` with `mt-2 max-w-2xl text-base md:text-lg leading-relaxed text-silicon-600`.

       **Image rendering (conditional on `milestone.image` being truthy):**
       Use `clsx` for variant-dependent image container classes:
       - `large` variant: `mt-6 w-full` — full-width image, `aspect-video` (16:9), `rounded-lg object-cover`.
       - `half` variant: `mt-6 grid grid-cols-1 md:grid-cols-2 gap-6` wrapping image and text side-by-side. On mobile, stacks with image on top.
       - `overlay` variant: `mt-6 relative w-full aspect-[3/2] rounded-lg overflow-hidden` — image fills container, text overlaid at bottom with `bg-gradient-to-t from-black/60 to-transparent` scrim for contrast.

       **Text-only fallback (no image):**
       When `milestone.image` is falsy, all three variants render the same: year, title, description stacked vertically. No empty image container. The `large` variant gets slightly larger year text (`text-6xl md:text-8xl`). The layout simply has more whitespace, which looks intentional with the generous padding.

       Since ALL 8 entries currently ship with no images, the text-only path is the default experience. It should look polished on its own — not like something is missing.

    5. **Main Timeline component:**
       ```typescript
       export function Timeline() { ... }
       ```
       - Same section wrapper as existing site pattern: `<section id="timeline" className="px-6 py-24">` with `<div className="mx-auto max-w-5xl">`.
       - Section heading: `<h2 className="text-2xl font-bold text-ink">Timeline</h2>`.
       - Connector wrapper: `<div className="relative mt-12">`.
       - **Connector line:** A `<div>` absolutely positioned on the left edge. `absolute left-[3px] top-0 bottom-0 w-[1.5px] bg-accent/30`. This thin accent-blue line runs the full height. Add `data-testid="timeline-connector"`.
       - Map `milestones` to `<TimelineEntry>` components.

    **What to import:**
    - `motion, type Variants` from `motion/react`
    - `clsx` from `clsx`
    - `easing` from `../../styles/motion`
    - `milestones` from `../../data/timeline`

    - Do NOT add pulse-ring or any expanding ring animation
    - Do NOT add a progress indicator (no filling line, no floating ring)
    - Do NOT store layout variant in data — compute from index only
    - Do NOT create separate files for layout variants — all in one Timeline.tsx

   - Clean up on unmount

   - Items below each group header are clickable, highlight when active (use accent/purple color for active state)
   - Lightweight dev-tool aesthetic: small text (text-xs for headers, text-sm for items), minimal padding, no heavy borders
   - Use lucide-react icons next to each nav item for visual identification:

     - Projects: `FolderOpen`, Papers: `FileText`
     - Skills: `Cpu`, Tooling: `Wrench`, Timeline: `Clock`, Coursework: `GraduationCap`
